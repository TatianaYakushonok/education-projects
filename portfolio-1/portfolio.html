Портфолио для фотографа

Верстка лендинга для фотографа. Навыки, портфолио, видео, прайс, форма для связи. С помощью JavaScript реализованы: светлая итемная темы лендинга, переключение с русского на английский язык, видео плеер для видео, переключение табов в секции портфолио по сезонам.
